# Claude Skills — PCP/PPCP Industrial

Skills para uso com Claude (claude.ai / Claude Code / Cowork), desenvolvidas para contexto de **Planejamento e Controle da Produção** industrial.

## Skills disponíveis

| Skill | Descrição |
|-------|-----------|
| `analise-indicadores-pcp` | OEE, eficiência, absenteísmo, aderência ao plano, Takt Time |
| `gestao-capacidade-gargalos` | TOC, balanceamento de linha, Yamazumi, priorização EDD/CR |
| `comunicacao-executiva` | Pirâmide de comunicação, templates para diretoria e gestores |
| `lideranca-influencia` | Influência sem autoridade, mapa de interesses, técnicas de alinhamento |
| `ia-aplicada-negocio` | IA aplicada ao PCP, Claude API, automação industrial, ROI |

## Como instalar

1. Baixar o arquivo `.skill` desejado
2. No Claude (Pro/Team): **Settings → Skills → Install from file**
3. Ou colar o conteúdo do `SKILL.md` diretamente nas **Project Instructions**

## Contexto

Desenvolvido para uso na **Patrimar Móveis Ltda** (Jaci/SP) e projetos **RitmoProd**.
